Setting Up Your Browse

This application uses the D3 library as a tool for visual representation of data. Only thing you need for this app is a browser with javascript enabled. You can find the source code here. Just extract it and open the html file. On Chrome, there is an issue with security and local files, so if you're running the app locally, you'll need to do "chrome.exe --disable-web-security" (i.e. with the Run command).
